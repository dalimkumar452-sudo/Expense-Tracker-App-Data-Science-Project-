# 💰 Expense Tracker App (Data Science Project)

An interactive application designed to help users manage personal finances through data-driven insights. 

## 🌟 Key Features
- **Transaction Entry:** Log daily expenses with categories and descriptions.
- **Data Analytics:** Real-time summary of total and average spending.
- **Visual Insights:** Dynamic Pie and Line charts to track spending patterns.
- **Storage:** Automatic data persistence using a local CSV engine.

## 🛠️ Tech Stack
- **UI:** Streamlit
- **Data Engine:** Pandas
- **Graphics:** Plotly
